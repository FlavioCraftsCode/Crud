Este é um projeto de API simplificada (CRUD) desenvolvido para fins de aprendizado, utilizando o ecossistema Spring Boot e persistência de dados no MySQL.

## 🚀 Tecnologias Utilizadas
* **Java 21**
* **Spring Boot 3.2.2**
* **Spring Data JPA** (Hibernate)
* **MySQL** (Banco de dados relacional)
* **Maven** (Gerenciador de dependências)
* **Lombok** (Para produtividade)

## 📋 Funcionalidades
A API permite realizar as quatro operações básicas:
- **C (Create):** Criar um novo usuário.
- **R (Read):** Listar todos os usuários cadastrados.
- **U (Update):** Atualizar nome e e-mail de um usuário existente.
- **D (Delete):** Remover um usuário do sistema.

## ⚙️ Como rodar o projeto localmente

### Pré-requisitos
- MySQL Server instalado e rodando.
- Java 17 ou superior.
- Maven.
